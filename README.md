# Next.js Project

## Quick Start

1. Install dependencies:  
   `npm install` or `yarn`
2. Run dev server:  
   `npm run dev` or `yarn dev`
3. Open:  
   `http://localhost:3000`

## Key Commands

- `dev`: Start development server
- `build`: Create production build
- `start`: Run production server
- `lint`: Run ESLint
